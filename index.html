Welcome to the file terminal

Your commands are:

ls

- 'ls' displays all the files and folders in the folder you're in

touch

- 'touch' filename filesize creates a file with a specified size in KB (defualt size is 1KB if left unspecified. ) - File and folder names must be a minimum of 1 and maximum of 12 characters. - Files also need a '.' followed by a 3 character extension.

mkdir

- 'mkdir' foldername creates a new folder in the current directory or a specified path (the full path is used for the name if you want to create a file in another directory)

rm

- 'rm' filename deletes a file in the current folder.

rmdir

- 'rmdir' foldername deletes a folder in the current folder.

cd

- 'cd' foldername changes to a folder in the current folder. - use ''..' as the folderName to change out of the current folder with cd. - e.g. (cd ..)
